## 2026-05-19 (3)

### Config schema — 5 new fields for better job search and apply quality

**`app.yaml` / `SearchConfig`:**
- `search.seniority_level` — target seniority tiers (e.g., `["mid", "senior"]`); prevents matching roles above/below intended level
- `search.min_company_rating` — hard floor on company rating (e.g., `3.5`); jobs below threshold can be filtered at scoring time

**`user.yaml` / `UserConfig`:**
- `education` — list of education entries; each has `degree`, `specialization`, `institution`, `graduation_year`, `grade`; fed to LLM for accurate form filling
- `experience.certifications` — list of certifications (e.g., `["ISTQB", "AWS SAA"]`); prevents LLM from guessing on cert questions
- `profile.urls` — `github`, `linkedin`, `portfolio` profile URLs; used for profile link fields on job application forms

**Models added:** `UserEducation`, `UserProfileUrls`  
**Templates updated:** `create_app_template()`, `create_user_template()`  
**Live configs updated:** `config/app.yaml`, `config/user.yaml`  
**Tests:** all 33 passing
