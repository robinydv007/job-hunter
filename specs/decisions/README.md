## Creating a New ADR
1. Copy `0000-template.md` to `NNNN-short-title.md` (kebab-case, sequential number)
2. Fill in Context, Decision, Alternatives, Consequences
3. Add a row to the table above
4. Commit: `git commit -m "docs(adr): NNNN — <title>"`
