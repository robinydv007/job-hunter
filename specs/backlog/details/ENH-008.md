# Enhancement: Anti-Blocking — Playwright Stealth Mode

## Problem
Playwright leaves detectable fingerprints:
- `navigator.webdriver = true` — the #1 bot detection signal
- Default Chrome user-agent contains automation markers
- Missing Chrome automation flags that real browsers have
- Consistent WebGL, canvas, and font fingerprints

Naukri uses bot detection services that check these signals.

## Solution

### 1. Use `playwright-stealth` package
```python
from playwright_stealth import stealth_async

async def scrape_jobs_from_page(page: Page, ...):
    await stealth_async(page)  # masks webdriver, adds realistic fingerprints
    await page.goto(url, ...)
```

### 2. Launch browser with stealth flags
```python
browser = await playwright.chromium.launch(
    headless=False,
    args=[
        "--disable-blink-features=AutomationControlled",
        "--disable-dev-shm-usage",
    ]
)
```

### 3. Set realistic user-agent
```python
context = await browser.new_context(
    user_agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 ..."
)
```

## Files Changed
- `pyproject.toml` — add `playwright-stealth` dependency
- `src/job_hunter/search/naukri.py` — apply stealth to page before navigation
- Browser launch code (wherever Playwright is initialized) — add stealth args

## Testing
- Run search, verify `navigator.webdriver` is `undefined` in browser console
- Verify user-agent matches a real Chrome version
- Verify no regression in job extraction accuracy

## Acceptance Criteria
- `navigator.webdriver` returns `undefined` (not `true`)
- User-agent matches a current Chrome desktop browser
- Job extraction still works correctly after stealth masking
- No additional dependencies beyond `playwright-stealth`
