# ENH-012: Fix Naukri search — replace placeholder base_url with actual URL

**Status**: resolved  
**Priority**: P1  
**Phase**: 1  
**Date**: 2026-04-06

## Problem

`src/job_hunter/search/naukri.py` had hardcoded placeholder string `"NAUKRI.base_url"` instead of actual URL, causing Naukri search to fail with invalid URL errors like:
```
Cannot navigate to invalid URL: NAUKRI.base_url/Technical%20Lead-jobs?k=Technical%20Lead&jobAge=1
```

## Solution

1. Create `src/job_hunter/config/job_boards/` package
2. Add `naukri.py` with `NaukriConstants` class containing `base_url`
3. Import `NAUKRI` from `config.job_boards.naukri` in search module

## Files Changed

- Added: `src/job_hunter/config/job_boards/__init__.py`
- Added: `src/job_hunter/config/job_boards/naukri.py`
- Modified: `src/job_hunter/search/naukri.py` — import NAUKRI and use `{NAUKRI.base_url}` in f-strings

## Result

Naukri search now correctly builds full URLs like:
- `https://www.naukri.com/Technical%20Lead-jobs?k=Technical%20Lead&jobAge=1`
- `https://www.naukri.com/Senior%20Software%20Engineer-jobs-2-in-bangalore?k=Senior%20Software%20Engineer&jobAge=1`

Search returns jobs successfully (19-45 jobs per run).
